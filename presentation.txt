SLIDE 2: The Problem
--------------------------------------------------------------------------------
[Slide Content]
* The ATS Bottleneck: Up to 75% of resumes are filtered out by automated screeners before reaching a human recruiter.
* Manual Personalization is Slow: Tailoring a resume and cover letter for 20+ roles takes hours of exhausting manual effort.
* Preparation Gaps: Candidates go into interviews unprepared for how their specific experience maps to the job description.
* Fragmentation: Job seekers use disconnected tools (one for PDF reading, one for writing, one for interview prep).

[Speaker Notes]
"The job search process today is broken. Job seekers face applicant tracking systems (ATS) that automatically screen resumes for specific keywords. Trying to manually customize a resume and write a unique cover letter for every single application is extremely time-consuming, taking hours per job. When candidates finally get an interview, they often struggle to articulate exactly how their specific background matches that company's unique needs. This creates a clear need for an integrated, automated helper."

SLIDE 4: The 5-Agent Sequence
--------------------------------------------------------------------------------
[Slide Content]
1. Gap Analyst:
   - Evaluates resume text against job requirements.
   - Extracts matching skills and flags missing ones.
2. Score Engine:
   - Calculates a compatibility score (0-100%) indicating alignment with the job.
3. Resume Rewriter:
   - Injects missing keywords and uses action verbs.
   - Never invents experience—maintains 100% truthfulness.
4. Cover Letter Writer:
   - Drafts a professional, 3-paragraph letter using only real resume facts.
5. Interview Coach:
   - Creates 7 tailored interview questions and strategies mapped to the user's background.

[Speaker Notes]
"Let's look under the hood at the 5-Agent Sequence. 
First, the Gap Analyst reads both inputs to map the skills. 
Second, the Score Engine outputs a compatibility score. 
Third, the Resume Rewriter rebuilds the resume bullets, boosting keywords and action verbs while keeping all facts honest. 
Fourth, the Cover Letter Writer crafts a concise, human-sounding pitch. 
Finally, the Interview Coach generates 7 highly targeted interview questions and answers, acting as a personal mentor."

SLIDE 5: Technical Highlights & Optimization
--------------------------------------------------------------------------------
[Slide Content]
* Client-Side PDF Parser: Uses 'pdfjs-dist' directly in the browser to extract plain text, keeping server endpoints lightweight and secure.
* Dual-Call Parallelization: 
  - Consolidates 5 agent steps into 2 structured API calls.
  - Slashes API response times and halves token usage.
  - Designed to fit comfortably within the Gemini API free tier limits.
* Smart Job Scraper:
  - Targets career pages directly via Cheerio parser.
  - Proxies through Jina.ai Reader for client-heavy or protected sites.
  - Recognizes auth-walls (like LinkedIn login prompts) and falls back to manual copy-paste.

[Speaker Notes]
"From a technical standpoint, we optimized for user experience and API efficiency. 
Instead of sending a heavy PDF file to the server, we use pdf.js to extract text directly on the client. 
To bypass rate limits and reduce latency, we don't call the LLM five separate times. Instead, we bundle the steps into two parallel requests—one for analysis (returning JSON) and one for writing (returning text separated by custom delimiters). 
We also implemented a smart web scraper that automatically handles JavaScript-heavy sites and falls back gracefully when sites require login walls."
